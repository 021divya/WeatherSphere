# 🌦️ WeatherSphere

WeatherSphere is a futuristic weather dashboard built using HTML, CSS, and JavaScript.

## ✨ Features

- Real-time weather data
- 3-hour & 5-day forecast
- Interactive weather maps
- Dynamic weather themes
- Animated charts
- Multi-city dashboard
- Travel planner
- Responsive glassmorphism UI
- Installable PWA support

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
- OpenWeatherMap API
- Leaflet.js
- Chart.js

## 🚀 Run Locally

1. Add your OpenWeatherMap API key in:

```js
js/config.js
```

2. Open the project using Live Server in VS Code.

## 📱 PWA Support

WeatherSphere can be installed like a native app on desktop and mobile devices.
